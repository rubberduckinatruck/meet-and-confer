Can We Afford It?

Separate sustainable dollars from one-time dollars.

This page should not be a calculator. It should be a decision dashboard that explains which funding sources can support recurring commitments and which are better suited for one-time investments.

Core rule: recurring raises and benefit contribution increases need recurring, unrestricted capacity. One-time carryforward can help with one-time payments, transitions, reserves, or short-term implementation costs.

      A simple database you can edit without rebuilding the site

      Recommended workflow: update the data file after each meeting, commit to GitHub, and the dashboard refreshes automatically.
